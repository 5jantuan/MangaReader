# MangaReader

Интеллектуальная платформа для автоматического перевода и публикации манги с использованием технологий Computer Vision, OCR и Machine Translation.

## О проекте

MangaReader — это веб-платформа, предназначенная для автоматизации процесса обработки и перевода графического контента. Система анализирует страницы манги, распознаёт текст внутри изображений, группирует OCR-фразы в речевые облака и выполняет автоматический перевод контента на различные языки.

Проект разрабатывается как дипломная работа и ориентирован на исследование возможностей интеграции технологий компьютерного зрения и нейронного машинного перевода в рамках единого pipeline обработки графического контента.

## Основные возможности

* Загрузка манги и глав
* OCR-распознавание текста на изображениях
* Определение координат текстовых областей
* Группировка OCR-фраз в speech bubbles
* Автоматический перевод контента через DeepL API
* Поддержка нескольких языков
* Хранение переводов в базе данных
* Веб-интерфейс для авторов и читателей
* Отображение перевода поверх страниц манги

Проект использует многослойную архитектуру:

* Domain
* Application
* Infrastructure

OCR реализован как отдельный микросервис, взаимодействующий с основным приложением через HTTP API.

## Pipeline обработки

1. Загрузка страниц манги
2. OCR-распознавание текста
3. Извлечение координат текстовых областей
4. Группировка OCR-фраз в Bubble
5. Машинный перевод Bubble
6. Сохранение BubbleTranslation
7. Отображение перевода пользователю

## Статус проекта

Проект находится в активной разработке.

Планируемые улучшения:

* Улучшенная группировка Bubble
* OCR review editor
* Автоматическое наложение перевода
* Поддержка PDF

Разработано в рамках дипломного проекта.
