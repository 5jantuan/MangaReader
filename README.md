
**MangaReader** — это учебный проект для чтения манги, разработанный на C# с использованием ASP.NET Core и Entity Framework Core. В проекте реализован просмотр страниц манги с фразами и их переводами, а также возможность добавления, редактирования и удаления фраз. Пользователи могут фильтровать фразы по языкам и работать с несколькими проектами одновременно благодаря архитектуре, разделённой на Web, Domain и Infrastructure. Для работы с базой данных используется PostgreSQL, а управление зависимостями и версиями осуществляется через NuGet. Проект позволяет изучить основы веб-разработки, работу с базой данных, паттерны репозитория и практическое использование Git для контроля версий.
