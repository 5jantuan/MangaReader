
**MangaReader** — это учебный проект для чтения манги, разрабатываемый на C# с использованием ASP.NET Core и Entity Framework Core. В будущем проект будет включать просмотр страниц манги с фразами и их переводами, возможность добавления, редактирования и удаления фраз, а также фильтрацию по языкам. Архитектура будет разделена на Web, Domain и Infrastructure, для работы с базой данных планируется использовать PostgreSQL. Проект создаётся для изучения веб-разработки, работы с базой данных, паттернов репозитория и практического применения Git для контроля версий.
